# MoCKA Orchestra Capability Matrix

| Agent      | Strengths                  | Weaknesses           | Cost Level | Role Primary |
|------------|----------------------------|----------------------|------------|--------------|
| Gemini     | Multi-modal, Logic, Fast   | Creative Nuance      | Low-Mid    | Logic/Fast   |
| ChatGPT    | Explanation, Education     | Fact Precision       | Mid        | Creative     |
| Claude     | Analysis, Long-form, Structure | Latency, Cost      | High       | Critic/Analyst|
| Perplexity | Search, News, Comparison   | Deep Reasoning       | Low-Mid    | Fact-Checker |
| Copilot    | Code, System Architecture  | General Chat         | Low        | Technical    |

## Consensus Protocol
1. **Selection**: Conductor selects 'Main Responder' based on prompt category.
2. **Review**: Other nodes provide 'Diff' (delta) feedback.
3. **Integration**: Conductor merges results into a single Process-aware answer.
