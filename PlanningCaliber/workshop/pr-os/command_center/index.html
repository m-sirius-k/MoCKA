GA4 設定手順
1. Google Cloud Console でサービスアカウントを作成
2. Google Analytics Data API を有効化
3. GA4 プロパティにサービスアカウントを閲覧者として追加
4. 認証情報JSONをダウンロード
5. python pros.py setup-ga <property_id> <creds.json> を実行
6. pip install google-analytics-data でライブラリをインストール
