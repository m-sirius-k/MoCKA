# AI Gate

Caliber内処理。入力コンテンツの校正・品質保証・原本確定を担う。

## 処理フロー

```
入力受付 → 前処理 → 校正エンジン → 整合性チェック → 最適化 → スコアリング → 確定判定 → KS_NNN発番
```

## 品質スコア閾値

| スコア | 処理 |
|--------|------|
| 0.8以上 | 自動確定 |
| 0.6〜0.8 | 手動承認待ち |
| 0.6未満 | 差し戻し |

## 実装予定ファイル

- `proofreader.py` — 誤字脱字・表記統一
- `integrity_check.py` — MoCKA蓄積知識との整合性検証
- `optimizer.py` — 構造化・見出し・要約自動生成
