How it works

AIとの対話を「資産」に変える、4ステップ。

セットアップは数分。あとはいつも通り Claude を使うだけで、会話が自動で蓄積されていきます。

1

Chrome に追加

Chrome Web Store から Orchestra をインストール。拡張アイコンがブラウザに表示されます。

2

Claude で会話開始

いつも通り Claude.ai を開いて会話するだけで、自動で記録が始まります。

3

右クリック → 協議

気になるテキストを選択して右クリック。複数 AI が並列で回答します。

4

CSV / JSON で持ち出し

蓄積した履歴は CSV / JSON でエクスポートし、外部ツールと連携できます。

FAQ

よくある質問。

導入前によくいただく質問をまとめました。その他のご質問はメールでお問い合わせください。

無料プランと Pro の違いは何ですか？
無料プランでは会話の保存・検索・エクスポートが使えます。Pro ではこれに加えて、多AI 並列協議と AI 別タイマー機能、協議プロンプトのカスタマイズ、メールサポートが利用できます。
ライセンスキーはいつ届きますか？
Stripe での購入完了後、自動システムが即時に noreply@nsjp.org からライセンスキーを送信します。通常 1 分以内に届きます。迷惑メールフォルダもご確認ください。
会話データはどこに保存されますか？
すべてのデータはお使いの Chrome ブラウザのローカルストレージに保存されます。外部サーバーへの送信は一切ありません。バックアップが必要な場合は CSV / JSON エクスポートをご利用ください。
One プランの「$8」はいつまでですか？
先着 500 名限定の特別価格です。枠がなくなり次第、通常価格の $10/月 に移行します。価格変更後も、既存ユーザーには変更前の価格が継続適用されます。
チーム利用や複数ライセンスは可能ですか？
現時点では 1 ライセンス = 1 ブラウザを想定しています。チーム向けのボリュームライセンスや管理機能については、今後のアップデートでの提供を検討しています。
