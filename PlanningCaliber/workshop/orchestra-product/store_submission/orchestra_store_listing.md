## Detailed Description

**AI forgets. Orchestra won't let it.**

Orchestra automatically saves every conversation on claude.ai — so you can search it, export it, and own it. Forever.

---

### What Orchestra does

**💾 Auto-save every conversation**
Every message you send and receive on claude.ai is captured automatically in your browser. No manual copying. No missed context.

**🔍 Full-text search across all sessions**
Find any decision, any piece of code, any idea you discussed — across every session, instantly. Filter by You / AI, sort by newest or oldest, search by keyword.

**📤 Export as CSV or JSON**
Download your entire conversation history in CSV (for Excel, Google Sheets) or JSON (for developers, data pipelines). No time limits. No subscription required to export.

**🔒 100% local. No server. Ever.**
Orchestra has no server. All data is written only to your browser's IndexedDB. External transmission is physically impossible. Uninstall and everything disappears — no residue anywhere.

---

### No API key required

Orchestra uses your existing browser sessions — the accounts you already have open. No need to purchase API keys from OpenAI or Anthropic. It works with whatever you're already using every day.
