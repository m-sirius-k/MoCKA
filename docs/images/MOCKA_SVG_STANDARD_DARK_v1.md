# MoCKA SVG Standard Dark v1.0

## Definition

MoCKA SVG Standard Dark v1.0 is the official visual specification for MoCKA structural diagrams, designed for dark-mode readability, academic reproducibility, and institutional consistency.

MoCKA SVG Standard Dark v1.0 は、ダークモード視認性、学術的再現性、制度的一貫性を目的として定義された、MoCKA 構造図版の公式視覚仕様である。

---

## Purpose

This specification defines a reusable standard form for MoCKA SVG diagrams across research papers, README files, governance documents, architectural figures, and system illustrations.

本仕様は、論文図版、README 図版、制度文書、構造図、エンジン図、層図に共通適用する再利用可能な MoCKA SVG 標準フォームを定義する。

The standard ensures:

1. High visibility on GitHub dark mode  
2. Academic reproducibility  
3. Long-term design consistency  
4. Readability over decoration  
5. Portable SVG architecture

## Layout Principles

Every diagram must clearly represent

central concept  
structural components  
flow relationships

Decoration such as gradients or glow effects should not be used.
