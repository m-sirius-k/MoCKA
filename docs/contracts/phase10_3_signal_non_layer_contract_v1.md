```

  確定された出口（接続先）を持たないことそのものが定義である。

  「存在しない」のではなく、「未生成の構造空白」（補正3の静的

  でも未設計の後段でもなく、Signalを経て初めて生成可能状態に
  なる動的領域として位置づけが確定した。
```
