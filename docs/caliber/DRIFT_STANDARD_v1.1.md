# Drift基準 v1.1
制定日：2026-04-01

## 基準値定義
| 状態 | Drift範囲 | 説明 |
|---|---|---|
| NORMAL | 0.0–1.0 | 正常動作 |
| WARNING | 1.0–2.0 | 軽度逸脱 |
| DANGER | 2.0–3.0 | 要介入 |
| CRITICAL | 3.0+ | 監査モード |

## 算出方法
直近N件のevents.csvから：
- ERROR含有率 × 重み0.6
- router介入率 × 重み0.4
= Drift score
