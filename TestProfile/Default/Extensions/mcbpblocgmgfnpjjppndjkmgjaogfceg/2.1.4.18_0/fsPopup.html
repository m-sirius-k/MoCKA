Welcome to FireShot!

FireShot works for any webpage, but with some exceptions, which are regulated by the browser's security policy.

Fortunately, only a few page types are protected from script execution: blank, extension, and startup pages.

You can try out any other web page, for example this page.

Action required

FireShot needs your approval to capture this resource. Please go to the options and check the Allow access for file URLs box.
