## エラー修正記録

### 修正1: タイムスタンプ型混在問題（M-S-06-d）
- **原因**: ledger.json の `timestamp` フィールドが float（Unix time）と str（ISO形式）の混在
- **修正**: 型チェックを追加し、同一型に統一してから比較

### 修正2: MCP server エンドポイント（M-S-09-c）
- **原因**: `/` が 404。実際のエンドポイントは `/health`, `/mcp`, `/agent/tools`
- **修正**: 複数エンドポイントを順番に試みる
